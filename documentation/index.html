Basic Map

To activate Maps Manager you have to specify attribute data-mapsmanager to be mapsmanager. In case if you are willing to create any specific map from available set you should specify the name of the map. Currently there are available four types of maps: bingmap, googlemap, mapbox and mapquest.
The code to create basic Google Map is shown below: 
<div data-mapsmanager="googlemap" style="height:200px"></div>
<!-- include once // -->
<script src="/path/to/jquery.min.js"></script>
<script src="/path/to/mapsmanager.min.js"></script>
<!-- // include once -->
We recommend you to put all your CSS in separate file using HTML tag <link>.

API Key

Every supported map has it own terminology for API key and its corresponding Maps Manager data- attribute:
  • Bing Map key or credentials corresponding attribute is data-bingmapkey;
  • Google Map API key attribute is data-googlemapkey;
  • MapBox accessToken corresponding attribute is data-mapboxkey;
  • MapQuest API key attribute is data-mapquestkey.
API key should be always specified for all map types except Google Map.
If your MapBox accessToken is 1234567890 then Maps Manager HTML code will look like: 
<div data-mapsmanager="mapbox" data-mapboxkey="1234567890" style="height:200px"></div>
<!-- include once // -->
<script src="/path/to/jquery.min.js"></script>
<script src="/path/to/mapsmanager.min.js"></script>
<!-- // include once -->
We recommend you to put all your CSS in separate file using HTML tag <link>.

Center and Zoom Level

By default map is centered to 54.79257, -4.25293 with zoom level equal to 6.
To change map center you need to set data-center attribute to specific latitude and longitude separated by comma: data-center="40.718119,-74.003906".
To change map zoom level set data-zoom attribute to the value you need: data-zoom="8".
Combining all together resulting HTML code will look like: 
<div data-mapsmanager="mapbox" data-mapboxkey="1234567890" data-center="40.718119,-74.003906" data-zoom="8" style="height:200px"></div>
<!-- include once // -->
<script src="/path/to/jquery.min.js"></script>
<script src="/path/to/mapsmanager.min.js"></script>
<!-- // include once -->
We recommend you to put all your CSS in separate file using HTML tag <link>.

Maps Switcher

By default to create Maps Manager map you should use mapsmanager as the value of the data-mapsmanager attribute. In this case maps switcher with all available map types sorted by the name will be created.
If you need only selected map types you should set the value of the data-maps attribute to the map types you required separated by comma. The order of the map type buttons will be the same as the order of map types you specified.
Lets assume you need googlemap, mapbox and bingmap in the order stated. The resulting code will look like: 
<div data-mapsmanager="mapsmanager" data-maps="googlemap, mapbox, bingmap" data-mapboxkey="1234567890" data-bingmapkey="1234567890" style="height:200px"></div>
<!-- include once // -->
<script src="/path/to/jquery.min.js"></script>
<script src="/path/to/mapsmanager.min.js"></script>
<!-- // include once -->
We recommend you to put all your CSS in separate file using HTML tag <link>.

Markers

Currently, there are two ways you can put one or more markers onto your map.

You should specify absolute path to the feed of your markers using attribute data-feed if you want your markers to be loaded every time map view bounds are changed. Use placeholder $bounds as a value of your selected parameter to notify your backend of current map view bounds. Placeholder $bounds will be substituted with the map view coordinates in the form of south latitude, west longitude, north latitude, east longitude. The coordinates of the map view will be appended to the path if placeholder $bounds will not be found. The map view coordinates example: 49.8138041489,-10.699362278,59.2510149736,2.1424407959.

You should specify absolute path to the static file of your markers using attribute data-feedonce if you want your markers to be loaded all at once.
Letâ€™s assume you decided to load all your markers at once. Your Map Manager HTML code will be: 
<div data-mapsmanager="mapbox" data-feedonce="js/markers.json" data-mapboxkey="1234567890" style="height:500px"></div>
<!-- include once // -->
<script src="/path/to/jquery.min.js"></script>
<script src="/path/to/mapsmanager.min.js"></script>
<!-- // include once -->
We recommend you to put all your CSS in separate file using HTML tag <link>.
The preview of the code above:
The markers feed or static file format should be as follows: 
[ [ latitude, longitude, '/marker/icon/path.image' ],
[ 51.115444, -0.715918, '/i/marker-icon.png' ],
...
[ 51.042102, -0.811518 ]]
The marker icon image can be omitted if default marker icon is set.
To set default marker icon you should specify the URL of the default marker icon using data-marker-icon attribute.
To set free space around every marker use data-space attribute. The value of the attribute will be used as the multiplier of the free space around the marker.

MapBox Map ID

Currently there is only one MapBox specific option. You are able to set basic map ID using attribute data-mapboxmapid. For more information see MapBox map ID option documentation. Additional map IDs to use could be found here.
